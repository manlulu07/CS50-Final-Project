Connect to esp8266


Please follow these instructions

    1. Copy the code under source code and paste it into your arduino IDE. Make sure you have all the reqired libraries installed.
    2. Upload the code on your Arduino IDE and after it's done uploading open the serial monitor ,copy the IP address desplayed and paste it below to connect to your esp8266
    3. When it's connected use the arrow keys on your keyboard to control your esp car
    4. Check under schematics above to get the schematics of the car
    5. Disconnect and log out when you are done
